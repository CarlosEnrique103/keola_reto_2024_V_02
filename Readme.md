# 👨🏻‍💻 Project

This is an app solve challenge of keOla.

## 🖥 Technologies & Languages

1. React
2. React Native
3. Expo
4. Redux
5. Lottie

## 🛠 Config

- You need to create .env with the values

```bash
# development
$ EXPO_PUBLIC_API_URL=https://inclubtest.com:2053
```

## 🛠 Installing

```bash
# development
$ yarn install
```

## 🍏 Running App

```bash
# development
$ yarn start
```
